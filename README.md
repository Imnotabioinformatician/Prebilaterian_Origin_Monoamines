# Prebilaterian origin of monoaminergic signalling

This repository contains the code, models, sequence resources, and selected source data associated with our study on the prebilaterian origin of monoaminergic signalling.

## Contents

This repository includes:

- **Figures submitted with the manuscript**
- **Hidden Markov Models (HMMs)** used for the identification of candidate receptors and enzymes
- **Seed alignment for the Ad-Hoc monoamine model** associated with HMM construction
- **Code used to reproduce figures related to receptor deorphanisation**
- **Raw data underlying receptor deorphanisation analyses**

## Data availability

Because of file size constraints, large raw datasets are **not stored in this GitHub repository**. These include:

- **Raw behavioural videos**
- **Raw mass spectrometry data**
- **Bioinformatic analysis files, including the sequence datasets and alignments used for phylogenetic analyses, are also deposited in Mendeley Data.**

While these datasets have been deposited in **Mendeley Data**. The repository record is currently not yet public, but this README will be updated with the accession details and direct link as soon as the dataset is released.

## Repository organisation

The repository is organised to provide the materials required to reproduce the main computational and pharmacological analyses presented in the manuscript. In particular, it includes:

- HMM files used for sequence identification
- Sequence resources used in the construction and validation of those models
- Scripts for processing and plotting receptor deorphanisation data
- Source datasets used to generate the corresponding figures

## Notes

This repository is intended to support transparency and reproducibility of the analyses presented in the manuscript. Some additional source data and heavy raw files are hosted externally, as described above.
